# =============================================================================
# Adversarial Command Corpus
# =============================================================================
#
# Real command variants AI coding agents generate when trying to install packages.
# Each line: EXPECTED_RESULT|COMMAND
#   BLOCK  — hook/wrapper should deny this
#   ALLOW  — hook/wrapper should allow this
#   PASS   — not a pip command, should pass through
#
# Run with: tests/run-corpus.sh
# =============================================================================
